Codefor港北の保育園マップ
---------------
GitHubで公開されている[さっぽろ保育園マップ](https://github.com/codeforsapporo/papamama)を元に[港北区　オープンデータを活用した子育て情報サービス検討プロジェクト](https://www.facebook.com/groups/kohoku.okp.since.2016/)によって開発された保育園マップです。
